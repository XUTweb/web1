段落 (Paragraph)

这是一个段落。段落用于组织文本内容，使文章更易读。

这是另一个段落。段落之间通常有空行分隔。

链接 (Links)

这是一个链接到 Example.com 的链接 (a 标签)

链接用于连接到其他页面或资源。target="_blank" 属性使链接在新标签页中打开。

图像 (Images)

图像用于在网页上显示图片。src 属性指定图片URL，alt 属性提供图片的替代文本。

div 和 span

这是一个 div 元素 (div 标签)。div 是块级元素，用于组织页面结构。

这是一段文字，其中包含一个 span 元素 (span 标签)。span 是行内元素，用于样式化文本的一部分。

表格 (Table)

表格用于组织数据。table 包含 tr (行)，tr 包含 th (表头) 和 td (单元格)。

表单 (Form)

表单用于收集用户输入。form 包含 label (标签) 和 input (输入字段)。
